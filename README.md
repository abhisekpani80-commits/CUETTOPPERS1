## 📋 Features
- ✅ NTA-authentic exam interface with question palette
- ✅ Real-time countdown timer with audio alerts
- ✅ Auto-save + session resume
- ✅ Negative marking (+5 / –1)
- ✅ AI-powered result analysis (Gemini API)
- ✅ Firebase Firestore integration
- ✅ Admin dashboard with code generation
- ✅ Language toggle (English / Hindi)
